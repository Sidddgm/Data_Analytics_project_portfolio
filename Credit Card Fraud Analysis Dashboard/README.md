# 💳 Credit Card Fraud Analysis Dashboard

## 📌 Project Overview
Developed an interactive Credit Card Fraud Analysis Dashboard using Python, SQL, and Power BI to analyze 75,000+ financial transactions. The project identifies fraudulent transactions, high-value payments, transaction trends, and customer spending behavior through dynamic KPIs and interactive visualizations.

---

## 🛠️ Tools & Technologies
- Power BI
- Python (Pandas, NumPy)
- SQL
- Microsoft Excel

---

## 📊 Dashboard Features
- Total Transactions
- Fraud Transactions
- Fraud Rate
- Total Transaction Amount
- Average Transaction Amount
- High Value Transactions
- Fraud Transactions by Hour
- Transactions by Amount Category
- High Value Fraud Analysis
- Top 10 Fraud Transactions
- Interactive Filters & Slicers

---

## 📈 Key Insights
- Analyzed over **75,000** transactions.
- Identified **121 fraudulent transactions**.
- Fraud rate was only **0.16%** of total transactions.
- Detected **762 high-value transactions**.
- Identified peak fraud hours and high-risk transaction categories.
- Built an interactive dashboard for fraud monitoring and business decision-making.
